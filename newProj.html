Pi Polaroid!

Lot of different approaches, and a lot of hiccups. This project was super fun as are the low quality prints

Little bit of soldering and circutry too!

I designed an interface for a raspberry pi camera with a touchscreen. The interface was such that after a picture is taken, the screen's diplay switches to the picture and allows the user to adjust the quality before deciding whether to print or not. This was done using pygame as the rendering of the touchscreen is much like that in games.
